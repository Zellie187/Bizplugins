# BizHub

BizHub is a secure client portal platform for WordPress.

## Features

- Secure Client Portal
- Company Management
- Service Requests
- Workflow Engine
- Document Management
- Notifications
- WooCommerce Integration
- Forminator Integration

## Requirements

- WordPress 6.6+
- PHP 8.2+
- MySQL 8+
