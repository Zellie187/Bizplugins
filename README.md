# BizHub Core Plugin

The BizHub Core Plugin provides the foundational services for the BizHub Platform.

---

# Version

0.1.0

---

# Status

Sprint 000 Complete

---

# Purpose

The plugin provides the core infrastructure required by all BizHub service modules.

Core responsibilities include:

- Bootstrap initialization
- Dependency injection
- Service registration
- Configuration management
- Event dispatching
- Logging
- Security services
- Module loading

---

# Minimum Requirements

- PHP 8.2+
- WordPress 6.x
- WooCommerce
- MySQL 8+
- Composer
