# Overview

BizHub is a modular WordPress platform designed to automate business administration services including:

- Company Registration
- Company Amendments
- Director Changes
- Compliance Services
- Accounting Services
- Document Management
- Workflow Automation
- Secure Client Portal
- Reporting
- REST API Integrations

The platform is designed as a collection of independent modules built around a common core.
